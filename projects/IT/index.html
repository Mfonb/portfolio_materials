Customers Segmentation Project.

October 2023


Brief

This project involved the querying of data from the TravelTide database to study customers behavior and to assign designated perks to customers of the TravelTide organisation. Guess the Number Game.

December 2024


Brief

The project focussed on developing a game which allow users to guess a randomly generated number by the App from a random range of numbers.The App produces feedbacks of rewards or retrials to the user instigated by the input provided the user.

The App returns a reward to the user if the user accurately guessed the number generated by the algorithm. If the number returned by the user as a guess was not in range asked, the user is asked to retry with a valid input.

The App also tracks the number of guesses made the user, along with the number of correct guesses. However, the number of correct guesses and the trials could be reset and count would begin at zero.

Python Weather App Project.

January 2025


Brief

The project was aimed at writing a code capable of getting specify weather details a towns (chosen by user) in the United Kingdom through an API (Application Programming Interface) source. The API source was the OpenWeather Map. The python built weather App first gets name of user, then welcomes to user to the App. The user is later prompted to input the name of the city of which wearther information the user seeks. The App returns the Current Temperature, Weather Condition, Wind Speed, and Humidity of the entered city. Finally, the App expresses gratitude to the user for using the App on closing.
